Concept

MESG is a decentralised infrastructure that allows Blockchain, Internet and Internet of things technologies to interact together.

Our mission is to reduce the gap between technologies, especially between Blockchain and Internet, by creating the MESG protocol.

This open source protocol is the common interface that any technology should implement to connect with the MESG infrastructure.

Users can connect any events from one technology to any actions on other technologies in just a few steps. See it like a Zapier or IFTTT for any technology and decentralised.
