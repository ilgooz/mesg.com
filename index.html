Intro

The goal of MESG is to create a Decentralised infrastructure in order to connect any Technologies from any Web technologies to Blockchain technologies, Internet of things, etc.

We want to reduce the gap between technologies, specially blockchain technologies and Internet. Nowadays it can be really complicated to execute some action on Internet in real time based on some Blockchain event or the other way around. That's why we created MESG.

MESG let you connect some events from one technology to some actions on one other technology in just few clicks. See it like a Zapier but for every technologies and decentralised.
